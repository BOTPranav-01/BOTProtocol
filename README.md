# BOTProtocol ⚡
**SkinForgeAI + RageSenseAI + StreamerAura**  
A cinematic AI-powered gaming desktop companion that turns your setup into a **main-character moment**.

## Features (MVP ✅)
- **SkinForgeAI**: Upload a Minecraft skin PNG → it becomes a cute desktop pet that follows your cursor.
- **RageSenseAI**: Mic-based rage meter (RMS) with smoothing + threshold.
- **StreamerAura**: OBS WebSocket triggers: flash an overlay source when rage spikes.
